## 리스트
* 리스트
    * 순서있는 값들의 모음
    * 값 추가시 인덱스(순서를 나타내는 번호표)가 0부터 자동 생성
    * 리스트 생성
        * 대괄호[] 사용시 자동생성
        ```py
        l = ['d','a','t','a']
        print(l) #['d','a','t','a']
        print(type(l))#<class 'list'>
        print(len(l)) # 4
        ```
    * 참고) 리스트의 모든 데이터 타입의 데이트럴 값으로 가질 수 있음
